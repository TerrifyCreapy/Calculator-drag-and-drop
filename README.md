<h1>Drag and drop calculator</h1>
<p>This project has only one page. There you can use two mods there are constructor and runtime</p>
<p><strong>Constructor</strong> mode used to move elements from left to right side in any order.</p>
<p><strong>RunTime</strong> mode used to run calculator.</p>
